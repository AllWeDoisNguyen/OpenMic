This is a social network for stand-up comedians and local comedy shows. This app was built on Ruby on Rails version 4.2.6. Uses Puma instead of Webrick for easier hosting on Heroku. Still needs more styling and refactoring. 

Also features some bootstrap for styling. 
Used Ajax, and Jquery for the search enging feature. 
You can check out the website at openmicmiami.co
